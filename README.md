Hecto E-Commerce

A fully-functional, modern React.js E-Commerce web app designed for seamless shopping experiences — with Add to Cart, Product Search, Multi-Page Navigation, and Dynamic Product Data fetched from the DummyJSON API.
Fully responsive and optimized for performance, Hecto delivers a real-world e-commerce experience with clean UI and smooth interactions.


---

⚡ Key Features

🛒 Add to Cart System — Add, remove, and manage products dynamically in your cart.

🔍 Search Functionality — Instantly filter and find products using real API data.

📦 Product Management — Browse and view detailed product info using DummyJSON API.

🧭 Multi-Page Routing — Implemented with React Router for a smooth page-switching experience.

💳 Buy / Checkout UI — Simulated purchase process to showcase complete e-commerce flow.

📱 Fully Responsive Design — Optimized for mobile, tablet, and desktop using Tailwind CSS / custom CSS.

⚙️ Component-Based Architecture — Clean, scalable, and easy to maintain React structure.

🚀 Getting Started

Clone and run it locally:

git clone https://github.com/Munna-Scriptz/Hecto-Ecommerce.git
cd Hecto-Ecommerce
npm install
npm start

Then open 👉 http://localhost:3000 in your browser.
